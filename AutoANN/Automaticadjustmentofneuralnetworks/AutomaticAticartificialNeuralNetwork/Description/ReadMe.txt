AutoANN是为了建立一个自动构建神经网络的模型，运行的库主要基于keras为前端，tensorflow为后端的构建神经网络的模型。

该神经网络自动化主要包括：
1、每一层的神经网络自动添加神元的个数，设计的主要方式为以16为基础自动添加神经元的个数；
2、神经网络的层数自动添加，神经网络在完成第一层的神经元的数量达到992时，
    自动添加一层隐藏层神经元，然后神经元的个数遵循第一层比第二层的神经元的个数多为前提
    自动组合，以此为32-16，48-32，48-16....的方式递归下去。
3、神经网络的参数的个数自动添加，添加的神经网络的参数，为0-16每次添加一个参数，当参数
    超过16个以后神经元的个数超过16个以后神经元的个数16.17.18.n，输入的n参数超过
    72个的时候选用9、10、11、12、13....的平方为主要的输入参数的神经网络。
    构造的神经网络的数据的最高维数为28的平方。

该神经网络的与测试函数为Ackley，然后以此增加神经元的个数，设计神经网络的主要目的是以输入
的参数的数量构建不同的神经网络，主要的神经网络模型如下：
    1、当神经网络的输入的参数在0-16之间选用的神经网络为全连接的神经网络进行拟合，拟合的层数不超过2层；
    2、但参数在16-72个时选用的神经网络为一维卷积神经网络，拟合的神经网络的层数不超过4层,实际使用两层，
      featuremap的个数最多不超过256个；
    3、参数超过72个时选用二维卷积神经网络，拟合的层数一般在四层。

理论上该神经网络为一个解决一类函数拟合的通用的神经网络的自动化调节神经网络，
通过神经网络的拟合过程和一般结论，可以在迅速的找到函数拟合的最佳的拟合精度和神经网络的结果。
